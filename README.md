# Endless Dungeon Prototype

This repository contains a lightweight browser game prototype inspired by your endless dungeon survival loop.

## Play locally

Open [index.html](index.html) directly in a browser, or run a simple static server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

## Controls

- `W`, `A`, `S`, `D` — move
- `Shift` — sprint
- `Space` — attack
- `F` — hold near a crate to open it
- `E` — summon protector
- `R` — activate shield

## Goal

Survive each wave, open crates, defeat the wave boss, and keep pushing for a higher score.

## GitHub Pages

To publish on GitHub Pages:

1. Create a GitHub repository.
2. Upload these files.
3. In the repository settings, enable GitHub Pages from the `main` branch.
4. Your site will be available at:
https://netapet.github.io/endless-dungeon/
